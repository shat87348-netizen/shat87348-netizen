- 操作系统：ESXi 6.5

**工作职责：**
- 🏗️ **架构搭建**：可视化框架技术选型搭建Openlayers+Cesium，这也是大多数成熟二三维框架的结构
- 💻 **主要研发**：独立研发90%的功能点，并且为满足最终需求克服各项难点
- 🚀 **现场部署**：现场部署因用户现场设备不同绝大多数版本比较低，所以采用将项目包成Docker镜像然后将Docker镜像再部署在VMware虚拟机中最终导出ESXi镜像的方式

**项目特色：**
- ✨ 使用Vue+Openlayers+Cesium的方式贯穿2D/3D的所有需求

- 📍 大量点位控制（几十万）：对于大量点位并且需要统一控制点位样式的，采用WebGL Points Layer方式添加，统一修改万级点位样式性能不受影响
- 🛠️ 工具箱：点击地图上的单位会弹出工具箱的功能，工具箱提供对单位的放大、缩小旋转、删除的功能
- 🗺️ 地图工具：研发了通用的比例尺、定位、测量、地图切换的工具
