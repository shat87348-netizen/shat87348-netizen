# 快速配置模板

复制以下内容，替换为你的实际信息：

## 需要替换的内容

### 基本信息
- `你的用户名` → 你的 GitHub 用户名（例如：`shatang`）
- `你的名字` → 你的真实姓名或昵称
- `your.email@example.com` → 你的邮箱地址

### 社交媒体链接
- LinkedIn: `https://linkedin.com/in/你的用户名`
- Twitter: `https://twitter.com/你的用户名`
- Blog: `https://your-blog.com`

### 项目信息
- 项目名称和描述
- 项目 GitHub 链接
- 使用的技术栈

### 技能标签
根据你的实际情况添加或删除技术栈徽章

---

## 快速替换命令（Linux/Mac）

如果你熟悉命令行，可以使用以下命令快速替换：

```bash
# 替换用户名
sed -i '' 's/你的用户名/YOUR_GITHUB_USERNAME/g' README.md

# 替换名字
sed -i '' 's/你的名字/YOUR_NAME/g' README.md

# 替换邮箱
sed -i '' 's/your.email@example.com/YOUR_EMAIL/g' README.md
```

---

## 检查清单

在提交之前，请确认：

- [ ] 所有 `你的用户名` 都已替换
- [ ] 所有 `你的名字` 都已替换
- [ ] 邮箱地址已更新
- [ ] 社交媒体链接已更新
- [ ] 项目信息已更新
- [ ] 技术栈徽章已更新（添加你会的，删除你不会的）
- [ ] 博客文章链接已更新（如果有）
- [ ] 所有链接都可以正常访问

## 主题颜色参考

如果你想更改主题颜色，可以使用以下主题：

### GitHub Stats 主题
- `radical` - 红色主题（默认）
- `dark` - 深色主题
- `vue` - Vue 绿色主题
- `tokyonight` - 东京之夜主题
- `onedark` - One Dark 主题
- `dracula` - Dracula 主题
- `nord` - Nord 主题

### 徽章样式
- `for-the-badge` - 大徽章（推荐）
- `flat` - 扁平样式
- `flat-square` - 扁平方形
